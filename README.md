# IronLog

A single-file, offline workout + meal + body tracker with a 12-month, 4-phase training and nutrition plan built in. No accounts, no server — all data is stored on your device.

**Live app:** `https://<your-username>.github.io/ironlog/` (after you enable GitHub Pages — see below)

## Features
- **Train** — per-set logging with a rest timer, an editable workout builder, and best-1RM-to-date per lift.
- **Meals** — a phase-based meal plan showing what to eat each day, plus per-meal logging with running daily macro totals.
- **Body** — weight, waist, body-fat, chest/arms/thighs, sleep, mood, energy, steps.
- **Charts** — lift progress, body trends, and calories vs target.
- **Settings** — goals, editable targets, phase selector, light/dark theme, and JSON backup/restore.

## Put it online in 2 minutes (GitHub Pages)
1. Go to <https://github.com/new> and create a repository named **ironlog** (Public).
2. On the new repo page, click **uploading an existing file** and drag in **`index.html`** (and this `README.md`). Commit.
3. Open **Settings → Pages**. Under *Build and deployment*, set **Source: Deploy from a branch**, **Branch: `main` / `root`**, and Save.
4. Wait ~1 minute, then reload the Pages settings — your live link appears at the top:
   `https://<your-username>.github.io/ironlog/`
5. Open that link on your phone and **Add to Home Screen** to install it.

## Install on your phone
- **iOS (Safari):** Share → Add to Home Screen.
- **Android (Chrome):** ⋮ → Add to Home screen.

It opens full-screen like a native app and works offline. Your data lives in the browser on that device — use **Settings → Export backup** periodically to save a copy.
